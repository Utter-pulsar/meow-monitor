# meow-monitor

Use this skill whenever you want to notify the user through the **赛博消息栏 / cyber rail**.

## Command

```bash
meow "codex" "$ARGUMENTS"
```

Clear the feed:

```bash
meow clear
```

Help:

```bash
meow help
```

## Guidance

Use it when:

- work is complete
- you need the user to look at something
- you are blocked and need input
- an important background task finished

If the CLI says the cyber rail is not visible right now, do not assume the user saw the message.
